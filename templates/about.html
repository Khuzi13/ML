Algorithm Details

A deep dive into the Random Forest Regressor powering our predictions.

What is Random Forest?

Random Forest is an ensemble learning method for classification, regression and other tasks that operates by constructing a multitude of decision trees at training time and outputting the class that is the mode of the classes (classification) or mean prediction (regression) of the individual trees.

In the context of our House Price Predictor, we use the RandomForestRegressor. It combines the predictions of many different decision trees to produce a more accurate and stable prediction of housing prices based on features like Area, Bedrooms, Bathrooms, Location, and Age.

Model Hyperparameters

After extensive hyperparameter tuning using Grid Search Cross Validation, we settled on the following optimal configuration for our model:

  • n_estimators: 200 (Number of trees in the forest)
  • max_depth: 15 (Maximum depth of each tree to prevent overfitting)
  • min_samples_split: 5 (Minimum number of samples required to split an internal node)
  • min_samples_leaf: 2 (Minimum number of samples required to be at a leaf node)

This configuration strikes a balance between capturing the complex non-linear relationships in the housing data and avoiding overfitting algorithms memory of the training data.

Performance Metrics

The model was trained on a comprehensive dataset of 15,000+ housing records across various regions. Performance on the unseen test set yielded:

  • R² Score (Coefficient of Determination): 0.945 — This means the model explains 94.5% of the variance in house prices.
  • Mean Absolute Error (MAE): 2.3 Lakhs — On average, our predictions are off by +/- 2.3 Lakhs.
  • Root Mean Squared Error (RMSE): 4.1 Lakhs.

Implementation Detail

Here is a simplified snippet of how the model is instantiated and trained using Python's scikit-learn library:


from sklearn.ensemble import RandomForestRegressor

# Initialize the model with optimized parameters
rf_model = RandomForestRegressor(
    n_estimators=200, 
    max_depth=15, 
    random_state=42
)

# Train the model
rf_model.fit(X_train, y_train)

# Make predictions
predictions = rf_model.predict(X_test)
